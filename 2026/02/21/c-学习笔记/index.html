变量

数据类型 变量名s = 变量初始值；

常量

  1. 宏常量

    define 常量名 常量值

    一般用于主函数之前

  2. const变量

    const 数据类型 变量名 = 变量值；

    一般用于函数之中

整型和实型（浮点型）

整型表示整数，实型表示小数

数据类型 取值范围
short（短整型） -2^15~2^15-1
int（整型） -2^31~2^31-1
long（长整型） -2^31~2^31-1
longlong（长长整型） -2^63~2^63-1
float（单精度实型） 7位有效数字
double（双精度实型） 15~16位有效数字

float数据在定义数值时一般在数字后加f

科学计数法

在c++中使用科学计数法时10用e表示，比如3e10表示3*10^10

字符型

用于表示单个字符

char 字符名 = ‘字符’；

字符型使用的是ASCII编码（整型），运行时会将字符转换为对应的ASCII码

通过（int)字符名 查看对应ASCII码

字符串型

  • char 变量名[] = ”字符串内容“;

  • string 变量名 = ”字符串内容“；

注意：使用string定义字符串要包含头文件string

布尔类型

表示条件判断真假的值，0对应假，1对应真

bool 变量名 = 真假

sizeof关键字

用来统计数据类型所占内存大小

short占2字节；

int占4字节；

long在windows系统占4字节，在Linux32位系统占4字节，Linux64位系统占8字节；

longlong占8字节；

float占4字节；

double占8字节；

char占1字节；

bool占1字节；

转义字符

常用转义字符

  • \n 换行

  • \\ 表示一个字符\

  • \t 水平制表符(保证每个输出内容占8格，整齐输出结果)

运算符

  • 算术运算符

    • 正号/加 +

    • 负号/减 -

    • 乘 *

    • 除 /

      两个整数相除，结果依然是整数，把小数部分删除，如果想要输出小数，一开始的两个数值要定义为浮点型，

      除数不能为0

    • 取模（余数） %

    • 递增(加一) ++

    • 递减（减一） —

      递增递减分为前置和后置，符号在前（前置）表示先赋值再运算，符号在后（后置）表示先运算再赋值

  • 赋值运算符

    • 赋值 =

    • 加等于 +=

      a += b即a+b = 相加结果，其他赋值运算符同理

    • 减等于 -=

    • 乘等于 *=

    • 除等于 /=

    • 取模等于 %=

  • 比较运算符

    • 等于 ==

    • 不等于 !=

    • 小于 <

    • 大于 >

    • 小于等于 <=

    • 大于等于 >=

  • 逻辑运算符

    • 非 !

    • 与 &&

    • 或 ||

选择结构

  • if语句

    如果有两种情况

    如果有多种情况

    if语句还可以进行嵌套，用于更精准的判断

  • 三目运算符

    表达式1 ? 表达式2 : 表达式3；

    如果表达式1为真，执行表达式2，反之执行表达式3

  • switch语句

循环结构

  • while语句

  • dowhile循环语句

  • for循环语句

     for(起始表达式，即定义变量;条件表达式;末尾循环体，即循环判断的改变)

     //函数
 }

    和if一样，任意循环都可以嵌套

跳转语句

  • break语句

    终止某一（内层）循环或是跳出switch选择

  • continue语句

    在循环中跳过某次满足条件的循环，执行下一次循环

  • goto语句

    无条件的跳转语句

    定义一个goto跳转标志，使用goto直接跳转

地址

存储内容的编号，用&查看

数组

数组中每个元素都是相同数据类型；

数组是由连续的内存位置组成的（后续指针会用到）；

方便统计数组所占内存（sizeof）；

可以获取数组在内存中的首地址；

  • 一维数组

    定义方法：

    1. 数据类型 数组名[数组长度]；

    2. 数据类型 数组名[数组长度] = {元素1，元素2，元素3，……}；

    3. 数据类型 数组名[] = {元素1，元素2，元素3，……}；

      数组的索引（下标）是从0开始，通过下标查找数组中的元素

  • 二维数组

    定义方法：

    1. 数据类型 数组名[行数][列数]；

    2. 数据类型 数组名[行数][列数] = {{元素1，元素2，……}，{元素3，元素4，……}}；

    3. 数据类型 数组名[行数][列数] = {元素1，元素2，……}；

    4. 数据类型 数组名[][列数] = {元素1，元素2，……}；

冒泡排序

最常用的排序方法，对数组内元素进行排序

  1. 比较相邻的元素，如果第一个比第二个大，交换；

  2. 对每一对相邻元素做相同的工作，执行完毕后最大值会排序到最右侧；

  3. 重复以上步骤，直到不需要比较；

函数

将一段经常使用的代码封装起来，减少重复代码

定义：返回值类型 （数据类型或void） 函数名（参数列表）

调用函数时直接使用函数名

函数的定义要在主函数之前，如果在主函数之后，在主函数之前要写函数的定义

函数的分文件编写（自己建立头文件）

  1. 创建后缀名.h的头文件

  2. 创建后缀名.cpp的源文件（存储函数）

  3. 在头文件写函数的声明

  4. 在源文件写函数的定义

值传递

形参改变，实参不改变

此时输出为

对于函数swap，电脑给a，b分配了地址，后续的交换改变的是a，b地址对应的值，而不是m，n地址对应的值，所以m，n的值不变

指针

通过指针间接访问内存，指针就是保存的地址

  • 地址编号是从0开始记录的十六进制数字

  • 可以利用指针变量保存地址

定义：数据类型 * 指针变量名；

让指针记录某个地址

解引用*p访问对应地址的内容

（sizeof）指针在32位系统占4字节，64位系统占8字节

空指针

指针变量指向内存中编号为0的空间的指针为空指针

空指针无法访问

野指针

指针变量指向非法的内存空间的指针为野指针

const和指针

  • cosnt修饰指针——常量指针

  • 指针修饰const——指针常量

  • const既修饰指针，又修饰常量——常量指针常量

  1. 常量指针

    常量指针的指向（地址）可以修改，指向的值不可以修改

  2. 指针常量

    指针常量的指向不可以修改，指向的值可以修改

  3. 常量指针常量

    指向和值都不可以修改

指针和数组

利用指针快速访问数组中的元素

指针和函数

由于值传递，函数中形参改变实参不改变

但利用指针作函数参数，可以修改实参的值

电脑给a，b分配了新地址，交换改变了新地址的值，但是这个值对应的是m，n的地址，也就是m，n的地址进行了交换

此时m,n的值成功交换了，因为m，n的地址进行了交换，所以m，n对应的值由于地址交换也进行了交换
